# Chinook-Music-Store-Analysis

📌 Project Overview

This project analyzes the Chinook Music Store Database using SQL to uncover insights related to:

Customer behavior
Sales trends
Genre performance
Customer retention
Regional market analysis

The goal is to provide data-driven business recommendations for improving revenue and customer retention.

🗂 Database Used

The Chinook database contains tables related to:

Customers
Invoices
Tracks
Albums
Artists
Genres
Employees
⚙️ Methodology
Data Understanding
Data Cleaning & Preparation
SQL Querying
Data Analysis & Visualization
Business Insights & Recommendations
📊 Key Insights
📈 Sales Performance
The USA generates the highest revenue.
Repeat customers contribute significantly to revenue growth.
🔁 Customer Retention
Long-term customers place almost 2× more orders than new customers.
Customer lifetime value increases with tenure.
🎸 Genre Analysis
Rock music contributes nearly 69% of total revenue.
Metal and Alternative/Punk are strong secondary genres.
⚠️ Churn Analysis
High churn was observed in markets like Italy and Denmark.
Retention strategies should vary by region.
🚀 Business Recommendations
Focus marketing on high-value regions like the USA and Canada
Introduce loyalty programs for repeat customers
Prioritize high-performing genres
Create retention campaigns for high-churn markets
