# Chinook-Music-Store-Analysis



## 📌 Overview
Analyzed the Chinook Music Store Database using SQL to identify insights related to sales, customer behavior, retention, and genre performance.

---

## 📊 Key Insights
- USA generated the highest revenue
- Repeat customers drove higher sales
- Rock music contributed ~69% of total revenue
- High churn observed in some European markets

---

## 🚀 Recommendations
- Focus on high-value markets
- Improve customer retention programs
- Promote top-performing genres
- Reduce churn through targeted campaigns

---

## 🛠 Tools Used
- SQL
- Data Analysis
- Data Visualization
